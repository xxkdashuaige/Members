## 如何添加个人介绍

### 第一步：Fork 仓库

点击右上角 **Fork** 按钮，把仓库 fork 到自己的 GitHub 账号下。

### 第二步：写个人介绍

在你的 fork 仓库里：
1. 进入 `个人介绍/` 文件夹
2. 复制 [`_模板.md`](个人介绍/_模板.md)，重命名为 `你的名字.md`
3. 按模板填写内容，删掉 `<!-- -->` 注释

### 第三步：把名字加上链接

回到本文件（`README.md`），找到自己所在的小组，把名字改成可点击的链接：

```markdown
[你的名字](个人介绍/你的名字.md)
```

### 第四步：提 PR

1. 改完后 commit，message 写：`添加 xxx 的个人介绍`
2. 回到原仓库，点击 **Pull Requests → New pull request**
3. 选择 **compare across forks**，选自己的 fork 分支
4. 创建 PR，等负责人 review 合入 

> 🤔 不会操作？在群里喊一声，队友帮你搞定。
